    2장. 데이터 전처리

    2.1 데이터 탐색(EDA)

    목적: 데이터 특성 파악, 패턴/이상 발견

    주요 탐색 항목

    항목 내용 방법
    기초 통계 평균, 중앙값, 표준편차, 분위수 describe()
    데이터 형태 행/열 수, 데이터 타입 shape, dtypes, info()
    결측치 확인 누락된 값 비율 isnull().sum()
    분포 확인 데이터 분포 형태 히스토그램, 박스플롯
    상관관계 변수 간 관계 상관행렬, 히트맵

    2.2 결측치 처리

    처리 방법

    방법 설명 적용 상황
    삭제 행/열 제거 결측 비율 낮을 때
    평균/중앙값 대체 수치형 대표값으로 채움 정규/비정규 분포
    최빈값 대체 범주형 최빈값으로 채움 범주형 데이터
    보간법 전후 값으로 추정 시계열 데이터
    예측 모델 ML로 결측값 예측 복잡한 패턴 존재 시

    2.3 이상치 처리

    탐지 방법

    방법 기준
    IQR Q1 - 1.5×IQR 미만, Q3 + 1.5×IQR 초과
    Z-score |Z| > 3
    박스플롯 시각적 확인

    처리 방법

    • 삭제: 명백한 오류일 때
    • 대체: 경계값/중앙값으로 변환
    • 변환: 로그 변환으로 스케일 조정
    • 분리: 별도 분석 또는 모델링

    2.4 데이터 통합

    방법 설명 Pandas 함수
    병합(Merge) 키 기준 조인 merge()
    연결(Concat) 행/열 방향 결합 concat()
    조인(Join) 인덱스 기준 결합 join()

    주의사항: 중복 제거, 키 불일치, 스키마 통일

    2.5 데이터 변환

    스케일링

    방법 공식 특징
    Min-Max (x-min)/(max-min) 0~1 범위, 이상치에 민감
    Standard (x-μ)/σ 평균 0, 표준편차 1
    Robust (x-Q2)/(Q3-Q1) 이상치에 강건

    인코딩

    방법 설명 적용
    레이블 인코딩 범주 → 정수 순서형 범주
    원-핫 인코딩 범주 → 이진 벡터 명목형 범주
    타겟 인코딩 범주 → 타겟 평균 고카디널리티

    기타 변환

    • 로그 변환: 왜도 감소, 정규분포 근사
    • 다항 변환: 비선형 관계 포착
    • 구간화(Binning): 연속형 → 범주형

    2.6 데이터 축소

    방법 설명
    차원 축소 PCA, LDA로 특성 수 감소
    특성 선택 중요 특성만 선택 (필터/래퍼/임베디드)
    샘플링 데이터 일부만 사용
    수치 정밀도 축소 float64 → float32

    2.7 데이터 분할

    기본 분할

    전체 데이터
├── Train (60~80%) : 모델 학습
├── Validation (10~20%) : 하이퍼파라미터 튜닝
└── Test (10~20%) : 최종 성능 평가

    분할 방법

    방법 설명 적용
    Hold-out 단순 비율 분할 대용량 데이터
    K-Fold CV K개 폴드로 교차검증 데이터 부족 시
    Stratified 클래스 비율 유지 분할 불균형 데이터
    Time Series Split 시간 순서 유지 시계열 데이터
