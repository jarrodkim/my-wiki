    6장. 머신러닝 연습문제

    6.1 유방암 예측 모델

    문제 정의

    항목 내용
    목표 유방암 악성/양성 분류
    유형 이진 분류
    데이터 sklearn의 breast_cancer 데이터셋
    특성 30개 (반지름, 질감, 둘레, 면적 등)

    프로세스

    데이터 로드
    ↓
EDA (분포, 상관관계 확인)
    ↓
전처리 (스케일링)
    ↓
Train/Test 분할
    ↓
모델 학습 (로지스틱, SVM, 랜덤포레스트 등)
    ↓
평가 (정확도, F1, AUC-ROC)

    핵심 포인트

    포인트 설명
    클래스 불균형 확인 악성/양성 비율 체크
    스케일링 필수 SVM, 로지스틱 회귀 적용 시
    평가 지표 의료 분야는 재현율(Recall) 중요
    특성 중요도 어떤 특성이 예측에 영향 주는지 분석

    6.2 AI 면접관, 지원자 합격 여부 예측하기

    문제 정의

    항목 내용
    목표 지원자 합격/불합격 예측
    유형 이진 분류
    특성 학력, 경력, 기술스택, 면접점수 등

    프로세스

    데이터 수집/로드
    ↓
EDA (합격률, 특성별 분포)
    ↓
전처리
├── 결측치 처리
├── 범주형 인코딩 (학력, 전공 등)
└── 수치형 스케일링
    ↓
모델 학습 (의사결정나무, 랜덤포레스트, XGBoost)
    ↓
평가 및 특성 중요도 분석

    핵심 포인트

    포인트 설명
    범주형 변수 처리 원-핫 인코딩 또는 레이블 인코딩
    해석 가능성 의사결정나무로 합격 기준 시각화
    공정성 성별, 나이 등 민감 변수 처리 고려
    임계값 조정 비즈니스 요구에 맞게 조정

    6.3 주택 가격 예측 모델

    문제 정의

    항목 내용
    목표 주택 가격 예측
    유형 회귀
    데이터 Boston Housing 또는 California Housing
    특성 면적, 방 수, 위치, 연식 등

    프로세스

    데이터 로드
    ↓
EDA
├── 가격 분포 확인 (왜도)
├── 특성 간 상관관계
└── 이상치 탐지
    ↓
전처리
├── 로그 변환 (타겟 변수)
├── 이상치 처리
└── 스케일링
    ↓
모델 학습 (선형회귀, 릿지, XGBoost)
    ↓
평가 (RMSE, MAE, R²)

    핵심 포인트

    포인트 설명
    타겟 변환 가격 분포가 왜곡되면 로그 변환
    다중공선성 VIF 확인, 릿지/라쏘로 해결
    특성 공학 방당 면적, 위치 기반 파생 변수
    앙상블 XGBoost, LightGBM으로 성능 향상

    6.4 쇼핑몰 고객 데이터 분석

    문제 정의

    항목 내용
    목표 고객 세분화 (군집화)
    유형 비지도학습 (군집)
    특성 구매 금액, 구매 빈도, 최근 구매일 등

    프로세스

    데이터 로드
    ↓
EDA (구매 패턴 분석)
    ↓
RFM 분석
├── Recency: 최근 구매일
├── Frequency: 구매 빈도
└── Monetary: 구매 금액
    ↓
전처리 (스케일링)
    ↓
K 결정 (엘보우, 실루엣)
    ↓
K-Means 군집화
    ↓
군집별 특성 분석 및 해석

    RFM 분석

    고객 세그먼트 예시
─────────────────
┌─────────────┬───────┬───────┬───────┐
│   세그먼트   │   R   │   F   │   M   │
├─────────────┼───────┼───────┼───────┤
│ VIP 고객    │  높음  │  높음  │  높음  │
│ 충성 고객   │  중간  │  높음  │  중간  │
│ 이탈 위험   │  낮음  │  낮음  │  중간  │
│ 신규 고객   │  높음  │  낮음  │  낮음  │
└─────────────┴───────┴───────┴───────┘

    핵심 포인트

    포인트 설명
    스케일링 필수 K-Means는 거리 기반
    K 선택 엘보우 + 실루엣 점수 종합
    군집 해석 각 군집의 비즈니스 의미 부여
    활용 타겟 마케팅, 이탈 방지 전략

    연습문제 요약

    문제 유형 핵심 기법 주요 평가
    유방암 예측 이진 분류 로지스틱, SVM Recall, AUC
    합격 예측 이진 분류 트리 기반 앙상블 F1, 특성 중요도
    주택 가격 회귀 선형회귀, XGBoost RMSE, R²
    고객 분석 군집화 K-Means, RFM 실루엣, 해석
