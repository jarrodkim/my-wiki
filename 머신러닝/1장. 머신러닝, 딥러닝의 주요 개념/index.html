    1.1 인공지능, 머신러닝, 딥러닝

    관계: AI ⊃ ML ⊃ DL

    • 인공지능(AI): 인간의 지능을 모방하는 기술 전체
    • 머신러닝(ML): 데이터로부터 패턴을 학습하는 AI의 하위 분야
    • 딥러닝(DL): 신경망 기반의 ML 기법

    1.2 핵심 용어

    용어 설명
    독립변수(X) 입력 데이터, 특성(feature)
    종속변수(Y) 예측 대상, 레이블(label)
    모델 X→Y 관계를 표현하는 함수
    학습 데이터로 모델 파라미터 최적화
    추론 학습된 모델로 새 데이터 예측

    1.3 학습 유형

    • 지도학습: 정답(레이블)이 있는 데이터로 학습 (분류, 회귀)
    • 비지도학습: 정답 없이 패턴 발견 (군집화, 차원축소)
    • 강화학습: 환경과 상호작용하며 보상 최대화

    1.4 머신러닝 프로세스

    문제정의 → 데이터수집 → 전처리 → 모델선택 → 학습 → 평가 → 배포

    1.5 과대적합과 과소적합

    구분 과대적합(Overfitting) 과소적합(Underfitting)
    현상 훈련 데이터에만 과하게 맞춤 패턴을 충분히 학습 못함
    원인 모델 복잡도↑, 데이터↓ 모델 복잡도↓, 학습 부족
    해결 정규화, 드롭아웃, 데이터 증강 모델 복잡도↑, 학습 시간↑

    1.6 딥러닝 주요 용어

    • 뉴런/노드: 연산 단위
    • 레이어: 뉴런의 집합 (입력층, 은닉층, 출력층)
    • 가중치(Weight): 학습되는 파라미터
    • 편향(Bias): 활성화 조절값
    • 활성화 함수: 비선형성 부여 (ReLU, Sigmoid 등)
    • 손실 함수: 예측 오차 측정 (MSE, Cross-Entropy)
    • 옵티마이저: 가중치 업데이트 방법 (SGD, Adam)
    • 에포크(Epoch): 전체 데이터 1회 학습
    • 배치(Batch): 한 번에 처리하는 데이터 묶음

    1.7 주요 모델 및 설명

    지도학습 모델

    분류(Classification)

    모델 설명
    로지스틱 회귀 시그모이드 함수로 확률 출력, 이진분류 기본 모델
    KNN K개 최근접 이웃의 다수결로 분류
    결정트리 특성 기준으로 데이터를 분할하는 트리 구조
    랜덤포레스트 여러 결정트리의 앙상블, 과적합 방지
    SVM 클래스 간 최대 마진 초평면 탐색
    나이브베이즈 베이즈 정리 기반, 특성 독립 가정

    회귀(Regression)

    모델 설명
    선형회귀 Y = wX + b, 연속값 예측의 기본
    릿지/라쏘 정규화 추가로 과적합 방지 (L2/L1)
    XGBoost 그래디언트 부스팅, 높은 성능의 앙상블
    LightGBM 대용량 데이터에 최적화된 부스팅

    비지도학습 모델

    모델 설명
    K-Means K개 중심점 기준 군집화
    DBSCAN 밀도 기반 군집화, 이상치 탐지 가능
    PCA 주성분 분석, 차원 축소
    오토인코더 신경망 기반 차원 축소/특성 학습

    딥러닝 모델

    모델 용도 설명
    DNN 범용 완전연결층 다층 구조
    CNN 이미지 합성곱으로 공간적 특성 추출
    RNN 시퀀스 순차 데이터 처리, 시계열/텍스트
    LSTM/GRU 시퀀스 장기 의존성 학습, RNN 개선
    Transformer NLP/범용 어텐션 기반, 병렬처리 가능
    GAN 생성 생성자-판별자 경쟁 학습
    VAE 생성 확률적 잠재공간 학습

    강화학습 모델

    모델 설명
    Q-Learning 상태-행동 가치함수 학습
    DQN 딥러닝 + Q-Learning
    PPO 정책 기반, 안정적 학습
    A3C 비동기 병렬 학습
