## 핵심개념

### 엔지니어의 기본 소양 점검

개발자/엔지니어로서 갖춰야 할 기본기를 점검합니다.

**기술적 기본기:**

| 영역 | 체크 항목 |
|------|----------|
| **타이핑** | 영문 타자 속도, 정확도, 특수문자 입력 |
| **터미널** | CLI 명령어 사용, 파일/폴더 조작 |
| **Git** | 버전 관리, 커밋, 푸시, 협업 |
| **문서화** | Markdown 작성, README 작성 |
| **검색** | 구글링, 공식 문서 활용, 에러 해결 |

**비기술적 기본기:**

| 영역 | 설명 |
|------|------|
| **문제 해결력** | 에러를 두려워하지 않고 해결책을 찾는 능력 |
| **자기 학습력** | 새로운 기술을 스스로 학습하는 능력 |
| **커뮤니케이션** | 기술적 내용을 명확히 전달하는 능력 |
| **끈기** | 막혀도 포기하지 않는 인내심 |
| **호기심** | "왜?"라고 질문하고 탐구하는 자세 |

**타이핑의 중요성:**

```
개발자의 하루:
- 코드 작성: 타이핑
- 검색: 타이핑
- 문서 작성: 타이핑
- 커뮤니케이션: 타이핑

타이핑 속도 향상 = 생산성 향상
특히 영문/특수문자 타이핑이 중요!
```

**권장 타이핑 속도:**

| 수준 | 영문 타수 (WPM) | 한글 타수 |
|------|----------------|----------|
| 초급 | 30~40 | 200~300 |
| 중급 | 50~70 | 400~500 |
| 고급 | 80+ | 600+ |

**코드 타이핑 연습:**

```python
# 이 코드를 직접 타이핑해보세요
def calculate_sum(numbers):
    total = 0
    for num in numbers:
        if num > 0:
            total += num
    return total

result = calculate_sum([1, -2, 3, -4, 5])
print(f"Sum: {result}")
```

```bash

# 커밋 수 확인

git log --pretty=format:"%h %ad %s" --date=short
```

#### 주요 질의응답 주제

- 취업 준비는 어떻게 하나요?

**학습 관련:**
- 예습/복습은 어떻게 하면 좋을까요?
- 모르는 것이 너무 많으면 어떻게 하나요?
- 다른 수강생들과 협업하나요?
- 개인 프로젝트는 어떻게 준비하나요?
