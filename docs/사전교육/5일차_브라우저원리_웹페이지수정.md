### HTML5 태그 구조

HTML(HyperText Markup Language)은 웹 페이지의 구조를 정의하는 마크업 언어입니다. 태그(Tag)를 사용하여 콘텐츠의 의미와 구조를 표현합니다.

**HTML 문서의 기본 구조:**

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>페이지 제목</title>
</head>
<body>
    <!-- 여기에 콘텐츠 작성 -->
    <h1>안녕하세요!</h1>
    <p>웹 페이지 내용입니다.</p>
</body>
</html>
```

**구조 설명:**

| 태그 | 설명 |
|------|------|
| `<!DOCTYPE html>` | HTML5 문서 선언 |
| `<html>` | HTML 문서의 루트(최상위) 요소 |
| `<head>` | 메타 정보 (문자셋, 제목, 스타일 등) |
| `<body>` | 실제 화면에 보이는 콘텐츠 |

**자주 사용하는 HTML5 태그:**

| 태그 | 용도 | 예시 |
|------|------|------|
| `<h1>`~`<h6>` | 제목 (1이 가장 큼) | `<h1>대제목</h1>` |
| `<p>` | 문단 | `<p>내용</p>` |
| `<a>` | 링크 | `<a href="url">링크</a>` |
| `<img>` | 이미지 | `<img src="image.jpg" alt="설명">` |
| `<div>` | 영역 구분 (블록) | `<div>영역</div>` |
| `<span>` | 인라인 영역 | `<span>텍스트</span>` |
| `<ul>`, `<ol>` | 목록 | `<ul><li>항목</li></ul>` |
| `<table>` | 표 | `<table><tr><td>셀</td></tr></table>` |
| `<form>` | 입력 양식 | `<form><input type="text"></form>` |
| `<button>` | 버튼 | `<button>클릭</button>` |

**시맨틱 태그 (의미있는 태그):**

```html
<header>헤더 영역 (로고, 네비게이션)</header>
<nav>내비게이션 메뉴</nav>
<main>메인 콘텐츠</main>
<article>독립적인 콘텐츠 (기사, 블로그 글)</article>
<section>콘텐츠 섹션</section>
<aside>사이드바, 부가 정보</aside>
<footer>푸터 영역 (저작권, 연락처)</footer>
```

**태그의 속성(Attribute):**

```html
<!-- id: 고유 식별자 (페이지에서 유일) -->
<div id="main-content">메인 콘텐츠</div>

<!-- class: 그룹 식별자 (여러 요소에 적용 가능) -->
<p class="highlight">강조된 문단</p>
<p class="highlight">이것도 강조</p>

<!-- style: 인라인 스타일 -->
<h1 style="color: blue;">파란 제목</h1>

<!-- href: 링크 주소 -->
<a href="https://google.com">구글로 이동</a>

<!-- src: 이미지/스크립트 경로 -->
<img src="photo.jpg" alt="사진 설명">
```

### DOM (Document Object Model)

DOM은 HTML 문서를 트리 구조로 표현한 것입니다. 브라우저는 HTML을 파싱하여 DOM 트리를 만들고, 자바스크립트로 이를 조작할 수 있습니다.

**DOM 트리 구조:**

```
document
└── html
    ├── head
    │   ├── meta
    │   └── title
    └── body
        ├── header
        │   └── h1
        ├── main
        │   ├── p
        │   └── div
        └── footer
```

**HTML과 DOM의 관계:**

```html
<body>
    <div id="container">
        <h1 class="title">제목</h1>
        <p>내용</p>
    </div>
</body>
```

위 HTML은 아래와 같은 DOM 트리가 됩니다:

```
body
└── div#container
    ├── h1.title ("제목")
    └── p ("내용")
```

**DOM 조작 (JavaScript):**

```javascript
// 요소 선택
const title = document.getElementById("main-title");
const items = document.getElementsByClassName("item");
const firstPara = document.querySelector("p");
const allParas = document.querySelectorAll("p");

// 내용 변경
title.textContent = "새로운 제목";
title.innerHTML = "<span>HTML 포함</span>";

// 스타일 변경
title.style.color = "red";
title.style.fontSize = "24px";

// 속성 변경
const link = document.querySelector("a");
link.setAttribute("href", "https://naver.com");

// 클래스 추가/제거
title.classList.add("highlight");
title.classList.remove("old-class");
title.classList.toggle("active");
```

**브라우저 렌더링 과정:**

```
1. HTML 파싱 → DOM 트리 생성
2. CSS 파싱 → CSSOM 트리 생성
3. DOM + CSSOM → 렌더 트리 생성
4. 레이아웃 계산 (위치, 크기)
5. 페인팅 (화면에 그리기)
```

### 크롬 개발자 도구 활용법

개발자 도구(DevTools)는 웹 페이지를 분석하고 디버깅하는 강력한 도구입니다.

**개발자 도구 열기:**
- 단축키: `F12` 또는 `Ctrl + Shift + I` (Windows) / `Cmd + Option + I` (Mac)
- 우클릭 → "검사" 메뉴

**주요 패널:**

| 패널 | 기능 |
|------|------|
| **Elements** | HTML/CSS 구조 확인 및 실시간 수정 |
| **Console** | JavaScript 실행 및 오류 확인 |
| **Sources** | 소스 코드 보기 및 디버깅 |
| **Network** | 네트워크 요청/응답 모니터링 |
| **Application** | 쿠키, 로컬 스토리지 등 확인 |

**Elements 패널 주요 기능:**

1. **요소 선택 도구** (왼쪽 상단 커서 아이콘)
   - 페이지에서 요소를 클릭하면 해당 HTML 코드로 이동

2. **HTML 실시간 편집**
   - 요소 더블클릭 → 내용 수정
   - 우클릭 → "Edit as HTML"

3. **CSS 스타일 확인/수정**
   - 오른쪽 Styles 패널에서 실시간 수정
   - 체크박스로 스타일 on/off

4. **Computed 탭**
   - 최종 적용된 스타일 확인

**Console 패널 활용:**

```javascript
// 요소 선택 후 조작
$0  // 현재 선택된 요소

// 간단한 테스트
console.log("Hello!");

// 요소 찾기
document.querySelector("h1")

// 텍스트 변경
document.querySelector("h1").textContent = "변경됨!"
```

**유용한 단축키:**

| 단축키 | 기능 |
|--------|------|
| `Ctrl + Shift + C` | 요소 선택 모드 |
| `Ctrl + F` | Elements에서 검색 |
| `Ctrl + L` | Console 지우기 |
| `Esc` | Console 토글 |

## 실습활동

### 실습 1: 간단한 웹 뼈대 코드 직접 입력하기

**목표:** HTML 기본 구조를 이해하고 직접 웹 페이지를 만들어봅니다.

#### Step 1: HTML 파일 생성

1. 텍스트 에디터(VS Code, 메모장 등)를 엽니다
2. 새 파일 생성 → `my_first_page.html`로 저장

#### Step 2: 기본 HTML 구조 작성

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>나의 첫 웹페이지</title>
    <style>
        /* CSS 스타일 */
        body {
            font-family: 'Malgun Gothic', sans-serif;
            max-width: 800px;
            margin: 0 auto;
            padding: 20px;
            background-color: #f5f5f5;
        }
        header {
            background-color: #333;
            color: white;
            padding: 20px;
            text-align: center;
            border-radius: 10px;
        }
        main {
            background-color: white;
            padding: 20px;
            margin-top: 20px;
            border-radius: 10px;
            box-shadow: 0 2px 5px rgba(0,0,0,0.1);
        }
        .highlight {
            background-color: yellow;
            padding: 2px 5px;
        }
        .profile-list {
            list-style-type: none;
            padding: 0;
        }
        .profile-list li {
            padding: 10px;
            border-bottom: 1px solid #eee;
        }
        footer {
            text-align: center;
            margin-top: 20px;
            color: #666;
        }
    </style>
</head>
<body>
    <header>
        <h1>안녕하세요!</h1>
        <p>나의 첫 번째 웹페이지입니다</p>
    </header>

    <main>
        <section>
            <h2>자기소개</h2>
            <p>저는 <span class="highlight">홍길동</span>입니다.</p>
            <p>현재 <strong>웹 개발</strong>을 배우고 있습니다.</p>
        </section>

        <section>
            <h2>프로필</h2>
            <ul class="profile-list">
                <li><strong>이름:</strong> 홍길동</li>
                <li><strong>직업:</strong> 학생</li>
                <li><strong>관심분야:</strong> 프로그래밍, AI, 데이터</li>
            </ul>
        </section>

        <section>
            <h2>좋아하는 것</h2>
            <ol>
                <li>코딩</li>
                <li>음악 감상</li>
                <li>독서</li>
            </ol>
        </section>

        <section>
            <h2>링크</h2>
            <p>
                <a href="https://github.com" target="_blank">GitHub 바로가기</a>
            </p>
        </section>
    </main>

    <footer>
        <p>&copy; 2024 My First Webpage. All rights reserved.</p>
    </footer>
</body>
</html>
```

#### Step 3: 브라우저에서 확인

1. 저장한 `my_first_page.html` 파일을 더블클릭
2. 브라우저에서 결과 확인

#### Step 4: 내용 수정해보기

다음 항목들을 본인 정보로 수정해보세요:
- [ ] 제목 변경
- [ ] 이름 변경
- [ ] 프로필 정보 수정
- [ ] 좋아하는 것 목록 수정
- [ ] 색상 변경 (CSS의 background-color 등)

### 실습 2: 개발자 도구로 뉴스 제목 바꿔보기

**목표:** 크롬 개발자 도구를 사용하여 웹 페이지의 요소를 실시간으로 수정해봅니다.

**중요:** 이 실습은 본인 브라우저에서만 임시로 보이는 변경입니다. 새로고침하면 원래대로 돌아갑니다. 실제 웹사이트가 변경되는 것이 아닙니다!

#### Step 1: 포털 사이트 접속

1. 크롬 브라우저로 [네이버](https://www.naver.com) 또는 [다음](https://www.daum.net) 접속

#### Step 2: 개발자 도구 열기

1. `F12` 키를 눌러 개발자 도구 열기
2. 또는 `Ctrl + Shift + I`

#### Step 3: 요소 선택 도구 사용

1. 개발자 도구 왼쪽 상단의 **요소 선택 아이콘** 클릭 (또는 `Ctrl + Shift + C`)
2. 페이지에서 변경하고 싶은 뉴스 제목 위로 마우스 이동
3. 해당 요소가 하이라이트되면 클릭

#### Step 4: HTML 수정하기

**방법 1: 더블클릭으로 수정**
1. Elements 패널에서 텍스트 부분 더블클릭
2. 원하는 텍스트 입력 (예: "홍길동이 세상을 바꾼다!")
3. Enter 키로 확정

**방법 2: Edit as HTML**
1. 요소 우클릭 → "Edit as HTML" 선택
2. HTML 코드 직접 수정
3. 바깥 영역 클릭하여 적용

**방법 3: Console에서 JavaScript로 수정**

```javascript
// 첫 번째 h2 태그의 텍스트 변경
document.querySelector("h2").textContent = "나의 뉴스 제목!"

// 특정 클래스를 가진 모든 요소 변경
document.querySelectorAll(".news_title").forEach(el => {
    el.textContent = "홍길동 특보!"
})
```

#### Step 5: 스타일 변경해보기

1. Elements 패널에서 요소 선택
2. 오른쪽 Styles 패널에서 스타일 수정

```css
/* 직접 추가하거나 수정 */
color: red;
font-size: 30px;
background-color: yellow;
border: 3px solid blue;
```

#### Step 6: 스크린샷 찍기

1. 변경된 화면을 스크린샷으로 저장
2. `Ctrl + Shift + P` → "Capture full size screenshot" 검색하여 전체 페이지 캡처 가능

#### 재미있는 실습 아이디어

1. **뉴스 제목을 본인 이름으로** - "홍길동, 드디어 개발자 되다!"
2. **검색창 placeholder 변경** - "무엇이든 물어보세요"
3. **로고 이미지 변경** - src 속성 수정
4. **버튼 색상 변경** - 배경색, 글자색 수정
5. **폰트 크기 키우기** - font-size 조절

## 추가 학습: 간단한 인터랙션 추가

HTML 파일에 JavaScript를 추가하여 클릭 이벤트를 만들어봅니다.

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>인터랙션 실습</title>
    <style>
        .box {
            width: 200px;
            height: 200px;
            background-color: skyblue;
            display: flex;
            align-items: center;
            justify-content: center;
            cursor: pointer;
            transition: all 0.3s;
        }
        .box:hover {
            transform: scale(1.1);
        }
    </style>
</head>
<body>
    <h1 id="title">클릭해보세요!</h1>

    <div class="box" id="colorBox">
        클릭하면 색이 바뀝니다
    </div>

    <p>클릭 횟수: <span id="counter">0</span></p>

    <script>
        // 변수 선언
        let count = 0;
        const colors = ['skyblue', 'coral', 'lightgreen', 'gold', 'violet'];

        // 요소 선택
        const box = document.getElementById('colorBox');
        const counter = document.getElementById('counter');
        const title = document.getElementById('title');

        // 클릭 이벤트
        box.addEventListener('click', function() {
            count++;
            counter.textContent = count;

            // 랜덤 색상 변경
            const randomColor = colors[Math.floor(Math.random() * colors.length)];
            box.style.backgroundColor = randomColor;

            // 제목 변경
            title.textContent = `${count}번 클릭했습니다!`;
        });
    </script>
</body>
</html>
```
