## 핵심개념

### 변수 (Variable)

변수는 데이터를 저장하는 공간입니다. 파이썬에서는 별도의 타입 선언 없이 값을 할당하면 자동으로 타입이 결정됩니다.

```python
# 변수 선언 및 할당
name = "홍길동"      # 문자열
age = 25            # 정수
height = 175.5      # 실수
is_student = True   # 불리언

# 변수 출력
print(name)         # 홍길동
print(age)          # 25
```

**변수 이름 규칙:**
- 영문자, 숫자, 언더스코어(`_`) 사용 가능
- 숫자로 시작할 수 없음
- 예약어(if, for, while 등) 사용 불가
- 대소문자 구분 (Name과 name은 다른 변수)

### 데이터 타입 (Data Type)

파이썬의 기본 데이터 타입을 이해합니다.

| 타입 | 설명 | 예시 |
|------|------|------|
| `int` | 정수 | `10`, `-5`, `0` |
| `float` | 실수 (소수점) | `3.14`, `-2.5` |
| `str` | 문자열 | `"Hello"`, `'Python'` |
| `bool` | 불리언 (참/거짓) | `True`, `False` |
| `list` | 리스트 (여러 값 저장) | `[1, 2, 3]` |

```python
# 타입 확인하기
x = 10
print(type(x))  # <class 'int'>

y = "Hello"
print(type(y))  # <class 'str'>
```

**타입 변환:**

```python
# 문자열을 정수로
num_str = "100"
num_int = int(num_str)  # 100

# 정수를 문자열로
age = 25
age_str = str(age)  # "25"

# 실수를 정수로 (소수점 버림)
pi = 3.14
pi_int = int(pi)  # 3
```

### 리스트 (List)

리스트는 여러 개의 값을 순서대로 저장하는 자료구조입니다.

```python
# 리스트 생성
fruits = ["사과", "바나나", "오렌지"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]  # 다양한 타입 혼합 가능

# 인덱스로 접근 (0부터 시작)
print(fruits[0])   # 사과
print(fruits[1])   # 바나나
print(fruits[-1])  # 오렌지 (마지막 요소)

# 리스트 수정
fruits[0] = "딸기"
print(fruits)  # ["딸기", "바나나", "오렌지"]
```

**리스트 주요 메서드:**

```python
fruits = ["사과", "바나나"]

# 요소 추가
fruits.append("오렌지")      # 끝에 추가: ["사과", "바나나", "오렌지"]
fruits.insert(1, "포도")     # 특정 위치에 추가: ["사과", "포도", "바나나", "오렌지"]

# 요소 삭제
fruits.remove("바나나")      # 값으로 삭제
del fruits[0]               # 인덱스로 삭제
popped = fruits.pop()       # 마지막 요소 꺼내기

# 기타
print(len(fruits))          # 리스트 길이
fruits.sort()               # 정렬
fruits.reverse()            # 역순
```

### 조건문 (if문)

조건에 따라 다른 코드를 실행합니다.

```python
# 기본 if문
age = 20

if age >= 18:
    print("성인입니다")
else:
    print("미성년자입니다")
```

```python
# if-elif-else (여러 조건)
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

print(f"학점: {grade}")  # 학점: B
```

**비교 연산자:**

| 연산자 | 의미 | 예시 |
|--------|------|------|
| `==` | 같다 | `a == b` |
| `!=` | 같지 않다 | `a != b` |
| `>` | 크다 | `a > b` |
| `<` | 작다 | `a < b` |
| `>=` | 크거나 같다 | `a >= b` |
| `<=` | 작거나 같다 | `a <= b` |

**논리 연산자:**

```python
# and: 둘 다 참이어야 참
if age >= 18 and age < 65:
    print("성인입니다")

# or: 하나라도 참이면 참
if day == "토요일" or day == "일요일":
    print("주말입니다")

# not: 참/거짓 반전
if not is_raining:
    print("우산이 필요 없습니다")
```

### 반복문 (for문)

동일한 작업을 여러 번 반복 실행합니다.

```python
# 리스트 순회
fruits = ["사과", "바나나", "오렌지"]

for fruit in fruits:
    print(fruit)
# 출력:
# 사과
# 바나나
# 오렌지
```

```python
# range() 함수 사용
for i in range(5):      # 0, 1, 2, 3, 4
    print(i)

for i in range(1, 6):   # 1, 2, 3, 4, 5
    print(i)

for i in range(0, 10, 2):  # 0, 2, 4, 6, 8 (2씩 증가)
    print(i)
```

```python
# 인덱스와 값 함께 사용
fruits = ["사과", "바나나", "오렌지"]

for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")
# 출력:
# 0: 사과
# 1: 바나나
# 2: 오렌지
```

### 실습 2: 간단한 숫자 계산기 만들기

**목표:** 사용자 입력을 받아 사칙연산을 수행하는 계산기를 만듭니다.

**실습 단계:**

1. 새 파일 생성: `calculator.py`
2. 아래 코드 작성:

```python
# 간단한 계산기 프로그램

print("=== 간단한 계산기 ===")

# 사용자 입력 받기
num1 = float(input("첫 번째 숫자를 입력하세요: "))
num2 = float(input("두 번째 숫자를 입력하세요: "))
operator = input("연산자를 입력하세요 (+, -, *, /): ")

# 조건문으로 연산 수행
if operator == "+":
    result = num1 + num2
    print(f"{num1} + {num2} = {result}")
elif operator == "-":
    result = num1 - num2
    print(f"{num1} - {num2} = {result}")
elif operator == "*":
    result = num1 * num2
    print(f"{num1} * {num2} = {result}")
elif operator == "/":
    if num2 != 0:
        result = num1 / num2
        print(f"{num1} / {num2} = {result}")
    else:
        print("오류: 0으로 나눌 수 없습니다!")
else:
    print("올바른 연산자를 입력해주세요.")
```

3. 실행 및 테스트:

```bash
python calculator.py
```

**예상 실행 결과:**
```
=== 간단한 계산기 ===
첫 번째 숫자를 입력하세요: 10
두 번째 숫자를 입력하세요: 3
연산자를 입력하세요 (+, -, *, /): +
10.0 + 3.0 = 13.0
```

### 실습 3: 이름 리스트 관리 프로그램 만들기

**목표:** 리스트와 반복문을 활용하여 이름을 추가, 삭제, 조회하는 프로그램을 만듭니다.

**실습 단계:**

1. 새 파일 생성: `name_manager.py`
2. 아래 코드 작성:

```python
# 이름 리스트 관리 프로그램

names = []  # 빈 리스트로 시작

print("=== 이름 리스트 관리 프로그램 ===")
print("명령어: add(추가), remove(삭제), list(목록), quit(종료)")
print()

while True:
    command = input("명령어를 입력하세요: ").lower()

    if command == "add":
        name = input("추가할 이름: ")
        names.append(name)
        print(f"'{name}'이(가) 추가되었습니다.")

    elif command == "remove":
        name = input("삭제할 이름: ")
        if name in names:
            names.remove(name)
            print(f"'{name}'이(가) 삭제되었습니다.")
        else:
            print(f"'{name}'을(를) 찾을 수 없습니다.")

    elif command == "list":
        if len(names) == 0:
            print("리스트가 비어있습니다.")
        else:
            print("\n--- 이름 목록 ---")
            for i, name in enumerate(names, 1):
                print(f"{i}. {name}")
            print(f"총 {len(names)}명")
            print()

    elif command == "quit":
        print("프로그램을 종료합니다.")
        break

    else:
        print("알 수 없는 명령어입니다. (add, remove, list, quit)")
```

3. 실행 및 테스트:

```bash
python name_manager.py
```

**예상 실행 결과:**
```
=== 이름 리스트 관리 프로그램 ===
명령어: add(추가), remove(삭제), list(목록), quit(종료)

명령어를 입력하세요: add
추가할 이름: 홍길동
'홍길동'이(가) 추가되었습니다.
명령어를 입력하세요: add
추가할 이름: 김철수
'김철수'이(가) 추가되었습니다.
명령어를 입력하세요: list

--- 이름 목록 ---
1. 홍길동
2. 김철수
총 2명

명령어를 입력하세요: quit
프로그램을 종료합니다.
```
